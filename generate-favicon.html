Quick Method: Use this design at favicon.io

Steps to Replace Favicon:

  1. Go to favicon.io/favicon-generator/
  2. Enter settings:
    • Text: AM
    • Background Color: #2563eb
    • Text Color: #ffffff
    • Font Family: Roboto or Montserrat
  3. Click "Download" to get the favicon package
  4. Extract the ZIP file
  5. Copy favicon.ico to replace src/app/favicon.ico
  6. Clear browser cache:
    • Press Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
    • Or hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
  7. Restart the dev server: npm run dev
